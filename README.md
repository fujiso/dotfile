## Dotfile

### About
Very very simple setup to use zsh + vim for python.  
I'll add some settings if I feel like it.  


### How to install

    $ git clone https://github.com/fujiso/dotfile.git
    $ cd ./dotfile
    $ git submodule init
    $ git submodule update

and install submodules...(see each repository) 

    $ sh ./setup.sh
